How To Install

In the default pluggin you will not get any style. but you can choose Different Style

open the Downloaded file kounty_main. you will see two folder
  • 1. dist
  • 2.docs
    • 1.Demos
    • 2.Documentation
in the dist folder you will find your plugin. Just add kounty.js after jQuery pluggin in your file. We give you two file normal file and its minified file so that you can use one of them Note: This plugin Must need jQuery. Till now we tested with latest jQuery version.

Default Structure

                        
                            
                        <div class="kounty-countdown demo-countdown" data-countdown-seperator="label"  data-timer-date="Feb-09-2019 16:15:20" data-countdown-message="Hurrah You are Done!">
                            <span class="countdown-days"></span>
                            <span class="countdown-hours"></span>
                            <span class="countdown-minutes"></span>
                            <span class="countdown-seconds"></span>
                        </div>

Enable Dots Style

Just copy and pase it to get our demo style. Besides this when you Enable dots Seperator By default we added a class countdown-dots to kounty-countdown. Now You can change any style. 

                        
                       .countdown-dots  span{
                        position: relative;
                        padding-right: 10px;
                        margin-right: 3px;
                        font-size: 20px;
                    }
                    .countdown-dots  span span{
                        display: none;
                    }
                    .countdown-dots span::after {
                        content: ":";
                        position: absolute;
                        top: 50%;
                        line-height: initial;
                        right: 0;
                        transform: translateY(-50%);
                        }
                     .countdown-dots span:last-child::after {
                        display: none; 
                    }
